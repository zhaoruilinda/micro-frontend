# micro-frontend
基于single-spa的微前端应用

# 查看效果

```
// 运行app1 vue应用
cd app1
npm run serve:micro
```

```
// 运行app2 vue应用
cd app2
npm run serve:micro
```

```
// 运行app3 react应用
cd app3
npm start
```

```
// 运行基座应用
cd layout
npm run serve
```
